Do not take screenshots or use browser preview tools (preview_screenshot, preview_snapshot, preview_inspect, etc.) after making UI changes.

**Why:** User prefers to review visual results themselves.

**How to apply:** After editing UI/frontend code, skip the verification workflow entirely. Make the changes and report what was done without launching or interacting with the preview.
